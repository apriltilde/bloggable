About

I hate static site generators they are so annoying and all you really want at the end of the day is just a simple blog that you dont have to go into the weeds editing html every time you just want to write something simple. To this end bloggable! Its as tumblr esque while still letting you edit whatever you would like. It works on free accounts so no worrying about cross site origin hotlinking something whatever errors it should just work... hopefully!

Github repo: bloggable

Features

  • infinite blog posts
  • WYSIWYG editing
  • your posts formated and messed with however you like
  • automatic image and gif hosting on catbox.moe
  • tags

Getting started

You need to make an account to start, use the register/login box to make and login to your account. You may use a fake domain to register with if you do not wish to use a real email address but this will make account recovery much harder (please try to remember your password).

Download the tumblr template.html file or the zonelets template.html and put it in your neocities

Where it says change to your iframe link here replace the iframe tag with the tag automatically generated below (will be an example userId until you log in or sign up)

<iframe src='https://arpilmyroomim.neocities.org/embed/index.html?userId=example'frameborder='none'></iframe>

I would reccomend either writing your own or downloading a template css file to make the blog look nicer there is a tumblr style css file style.css and a zonelets style css file style.css remember to rename these files to styles.css when you add them to your neocities.
